## Getting Started

[Dotbot][dotbot] will handle entire installation process. Just:

```bash
cd ~
git clone https://github.com/marekkaczkowski/dotfiles.git
cd dotfiles
./install
```

Note that the `install` script is idempotent - running it multiple
times has no effect.
